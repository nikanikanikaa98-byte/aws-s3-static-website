# AWS S3 + CloudFront Static Website Project

## Overview
This project demonstrates how to host a static website using Amazon S3 and deliver it securely through Amazon CloudFront.

The website is stored in an S3 bucket and distributed globally using CloudFront with HTTPS support.

---

## Architecture
User → CloudFront → Amazon S3 Static Website Hosting

---

## AWS Services Used
- Amazon S3
- Amazon CloudFront

---

## Features
- Static website hosting
- Public website delivery through S3
- Content delivery through CloudFront
- HTTPS access via CloudFront
- Simple HTML/CSS user interface
